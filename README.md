About
-------------

I've written the following subroutine

- GCD
- Modulo Inverse (FISH)
- Square And Multiply
- Generator

I've used these to implement the following

- Diffie-Hellman key-exchange
- El Gamal
- RSA
- El Gamal Digital Signature
- AES


## Usage 

Compile the code by, simply:

	$ make

Run the program by


	$ make run
